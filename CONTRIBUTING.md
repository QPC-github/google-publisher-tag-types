# How to contribute

The Google Publisher Tag (GPT) type definitions are automatically generated, so we're not able to accept any external modifications at this time.

## Providing feedback

We value input from the comunity and encourage you to use GitHub Issues to provide feedback, report bugs, or suggest changes you'd like to see made.

Please keep in mind that this repository is specifically for GPT type definitions. If you have GPT feedback that is not directly related to the contents of this repository, please submit it using the **Send feedback** button at the top right of [this page](https://developers.google.com/publisher-tag/).
